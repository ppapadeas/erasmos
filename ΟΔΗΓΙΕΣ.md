# Οδηγίες Διαχείρισης Website - Erasmos Frontistirio

Βαγγέλη, παρακάτω θα βρεις αναλυτικές οδηγίες για να πάρεις τον έλεγχο του site, να συνδέσεις το domain σας, και να κάνεις αλλαγές στο περιεχόμενο.

---

## 1. Δημιουργία Λογαριασμού GitHub

Αν δεν έχεις ήδη λογαριασμό:

1. Πήγαινε στο https://github.com/signup
2. Φτιάξε δωρεάν λογαριασμό (π.χ. `erasmosfront`)
3. Στείλε μου το username σου για να σε κάνω collaborator στο repository

---

## 2. Μεταφορά του Repository στον λογαριασμό σου

Αφού σε κάνω collaborator, θα μπορείς να κάνεις αλλαγές. Εναλλακτικά, μπορώ να σου κάνω **transfer** ολόκληρο το repository στον δικό σου λογαριασμό:

1. Φτιάξε τον λογαριασμό σου στο GitHub
2. Στείλε μου το username
3. Εγώ θα κάνω transfer από τις ρυθμίσεις του repo
4. Εσύ αποδέχεσαι το transfer

Μετά το transfer, το site θα είναι στο: `https://[username].github.io/erasmos/`

---

## 3. Σύνδεση του Domain (erasmosfront.gr)

### Βήμα Α: Ρυθμίσεις DNS (στον πάροχο domain)

Μπες στον πάροχο που έχεις καταχωρημένο το `erasmosfront.gr` (π.χ. Papaki, TopHost, κλπ) και πρόσθεσε τα εξής DNS records:

**A Records** (και τα 4):
```
@ → 185.199.108.153
@ → 185.199.109.153
@ → 185.199.110.153
@ → 185.199.111.153
```

**CNAME Record** (για το www):
```
www → [username].github.io.
```

Αντικατέστησε το `[username]` με το GitHub username σου.

### Βήμα Β: Ρυθμίσεις στο GitHub

1. Πήγαινε στο repository → **Settings** → **Pages**
2. Στο πεδίο **Custom domain** γράψε: `erasmosfront.gr`
3. Πάτα **Save**
4. Τσέκαρε το **Enforce HTTPS** (μπορεί να χρειαστεί μερικά λεπτά μέχρι να ενεργοποιηθεί)

### Βήμα Γ: Ενημέρωση αρχείου ρυθμίσεων

Στο αρχείο `astro.config.mjs` άλλαξε:
```javascript
export default defineConfig({
  site: 'https://erasmosfront.gr',
  // αφαίρεσε τη γραμμή base: '/erasmos',
});
```

Και πρόσθεσε ένα αρχείο `public/CNAME` που να γράφει μόνο:
```
erasmosfront.gr
```

> Η αλλαγή DNS μπορεί να πάρει μέχρι 24 ώρες μέχρι να ενεργοποιηθεί πλήρως.

## 4. Πώς να Αλλάζεις Περιεχόμενο

### Μέθοδος Α: Απευθείας από το GitHub (εύκολη - χωρίς εγκατάσταση)

1. Μπες στο repository στο GitHub
2. Πλοηγήσου στον φάκελο `src/components/`
3. Κάνε κλικ στο αρχείο που θέλεις να αλλάξεις
4. Πάτα το εικονίδιο μολυβιού (Edit)
5. Κάνε τις αλλαγές σου
6. Πάτα **Commit changes**
7. Το site ενημερώνεται αυτόματα σε ~1 λεπτό!

### Μέθοδος Β: Τοπικά στον υπολογιστή σου (για πιο σύνθετες αλλαγές)

Χρειάζεσαι εγκατεστημένα:
- [Git](https://git-scm.com/downloads)
- [Node.js](https://nodejs.org/) (έκδοση 20+)

```bash
# Κατέβασε το project
git clone https://github.com/[username]/erasmos.git
cd erasmos

# Εγκατέστησε τα dependencies
npm install

# Ξεκίνα τοπικό server (βλέπεις αλλαγές ζωντανά)
npm run dev

# Ανέβασε αλλαγές
git add .
git commit -m "Περιγραφή αλλαγής"
git push
```

Μετά το `git push`, το site ενημερώνεται αυτόματα.

## 6. Παραδείγματα Συνηθισμένων Αλλαγών

### Αλλαγή τηλεφώνου

Στο `src/components/Contact.astro`, βρες:
```html
<a href="tel:+302102582744">210 258 2744</a>
```
Και άλλαξέ το στο νέο νούμερο.

**Πρόσεξε:** Το τηλέφωνο εμφανίζεται και στο `src/components/Footer.astro` - άλλαξέ το και εκεί.

### Προσθήκη νέας ανακοίνωσης

Στο `src/components/News.astro`, βρες τον πίνακα `announcements` και πρόσθεσε ένα νέο αντικείμενο:
```javascript
{
  title: 'Τίτλος ανακοίνωσης',
  excerpt: 'Σύντομη περιγραφή της ανακοίνωσης...',
  date: 'Μήνας',
  category: 'Ανακοίνωση',
  color: 'blue',  // επιλογές: blue, gold, red
},
```

### Αλλαγή ωρών μαθημάτων

Στο `src/components/Curriculum.astro`, βρες τα αντίστοιχα αντικείμενα και άλλαξε τις τιμές `hours`.

### Αλλαγή λογότυπου

Αντικατέστησε το αρχείο `public/images/logo.png` με το νέο λογότυπο (κράτα το ίδιο όνομα αρχείου).

## 7. Σημαντικές Σημειώσεις

- **Αυτόματο deploy:** Κάθε φορά που κάνεις αλλαγή και πατάς "Commit", το site ενημερώνεται αυτόματα μέσα σε 1-2 λεπτά
- **Κόστος:** Το GitHub Pages είναι **εντελώς δωρεάν** - δεν πληρώνεις hosting
- **Backup:** Το GitHub κρατάει ιστορικό όλων των αλλαγών - μπορείς πάντα να γυρίσεις σε παλιότερη έκδοση
- **SSL:** Το HTTPS (πιστοποιητικό ασφαλείας) παρέχεται δωρεάν αυτόματα
- **Φόρμα επικοινωνίας:** Η φόρμα χρησιμοποιεί το Formspree (δωρεάν μέχρι 50 μηνύματα/μήνα). Αν χρειάζεσαι περισσότερα, μπορείς να φτιάξεις λογαριασμό στο https://formspree.io και να αντικαταστήσεις το URL στο `Contact.astro`
